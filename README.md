## 🚀 Features

- **Professional Aesthetics:** Custom Earth-toned color palette (Forest Green & Stone Beige).
- **TypeScript Powered:** Strongly typed components and data structures for robust code.
- **Responsive Design:** Fully optimized for mobile, tablet, and desktop using Tailwind CSS.
- **Glassmorphism UI:** Modern backdrop-blur effects and subtle interactive animations.
- **Dynamic Project Gallery:** Scalable architecture for showcasing multiple software projects.

## 🛠️ Tech Stack

- **Frontend:** React 18, TypeScript, Tailwind CSS
- **Build Tool:** Vite
- **Deployment:** Vercel / GitHub Pages
